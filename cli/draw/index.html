ttt draw image#

Usage:

ttt draw image [OPTIONS] FILE | URL

Draw a picture provided by the given FILE or URL.

Options:

Name Type Description Default
--invert boolean Draw using inverted colors. False
--braille boolean Draw using braille dots instead of block characters. False
-o,
--outline		 "none" | "soft" | "hard" | "shadow" Draw using an outline effect. This option can be repeated. None
--help boolean Show this message and exit. False

ttt draw icon [OPTIONS]

Icons by PiiiXL (https://piiixl.itch.io).

--design		 text Icon design number or alias. Use command 'list icons' to see all icons. 31

--braille boolean Draw using braille dots instead of block characters. False
-o,
--outline		 "none" | "soft" | "hard" | "shadow" Draw using an outline effect. This option can be repeated. None

ttt draw atlas#

Usage:

ttt draw atlas [OPTIONS] FILE | URL

Draw a sprite from the sprite atlas provided by the given FILE or URL.

Options for width, height, offsets, and gaps are used to define the sprite layout, and the index option specifies a particular sprite to draw. If no index is provided, all sprites are drawn consecutively.

Options:

Name Type Description Default
-w,
--width		 integer Width of each sprite in the atlas (in pixels). required
-h,
--height		 integer Height of each sprite in the atlas (in pixels). required
-ox,
--offset-x		 integer Horizontal offset from the top-left corner of the atlas to the first sprite (in pixels). Default is 0. 0
-oy,
--offset-y		 integer Vertical offset from the top-left corner of the atlas to the first sprite (in pixels). Default is 0. 0
-gx,
--gap-x		 integer Horizontal gap between sprites in the atlas (in pixels). Default is 0. 0
-gy,
--gap-y		 integer Vertical gap between sprites in the atlas (in pixels). Default is 0. 0
-i,
--index		 integer Index of the sprite to draw (0-based). If not provided, all sprites are drawn consecutively. 0
--invert boolean Draw using inverted colors. False
--braille boolean Draw using braille dots instead of block characters. False
-o,
--outline		 "none" | "soft" | "hard" | "shadow" Draw using an outline effect. This option can be repeated. None
--help boolean Show this message and exit. False
