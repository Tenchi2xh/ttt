ttt frame#

Usage:

ttt frame [OPTIONS] [TEXT]

Renders TEXT with a specified font inside a frame.

Input can also be provided through a pipe.

Options:

Name Type Description Default
-V,
--verbatim		 boolean Don't format input and display it verbatim inside the frame. False
-P,
--frame-perfect		 boolean Force the frame width to be at a multiple of 8 to align with the original pixel art. False
-F,
--full-width		 boolean Use as much width as possible. False
-p,
--padding		 integer Padding (in pixels) between the frame and the contents. 0
-L,
--left-icon		 text Icon to display on the left. Use command 'list icons' to see all icons. None
-R,
--right-icon		 text Icon to display on the right. Use command 'list icons' to see all icons. None
-d,
--design		 text Frame design number or alias. Use command 'list frames' to see all frames. 27
-f,
--font		 font Specify the font to use for rendering the text. Use command 'list fonts' to see all fonts. Default is 'monogram'. monogram
--invert boolean Draw using inverted colors. False
--braille boolean Draw using braille dots instead of block characters. False
-o,
--outline		 "none" | "soft" | "hard" | "shadow" Draw using an outline effect. This option can be repeated. None
--help boolean Show this message and exit. False
