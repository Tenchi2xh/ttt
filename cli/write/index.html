ttt write#

Usage:

ttt write [OPTIONS] [TEXT]

Renders TEXT with a specified font or list available fonts.

Input can also be provided through a pipe.

Options:

Name Type Description Default
-f,
--font		 font Specify the font to use for rendering the text. Use command 'list fonts' to see all fonts. Default is 'monogram'. monogram
--invert boolean Draw using inverted colors. False
--braille boolean Draw using braille dots instead of block characters. False
-o,
--outline		 "none" | "soft" | "hard" | "shadow" Draw using an outline effect. This option can be repeated. None
--help boolean Show this message and exit. False
